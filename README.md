<h1>Site da Etec Zona Leste</h1>

<h2>Views</h2>

<h3>Landing</h3>
<p>Página inicial. Te leva para o resto do site</p>

<h3>Eventos</h3>
<p>Coordenadores podem postar eventos, colocando título, texto, horário e imagem. CRUD completo</p>

<h3>Cursos</h3>
<p>Professores e coordenadores podem criar cursos. Alunos podem acessar seus cursos out remover matrícula. CRUD completo</p>

<h3>Atividades</h3>
<p>Professor de um curso / Alunos matriculados podem acessar suas atividades. CRUD completo</p>

<h3>Conta</h3>
<p>Alunos podem fazer matrículas em cursos</p>

<h3>Fallback</h3>
<p>Página que te retorna para o landing</p>

<h2>Como Rodar</h2>
<p>Coloque o projeto na pasta xampp/htdocs, crie a database 'cesarmfetec', rode as migrations com o comando 'php artisan migrate' e rode o site com o comando 'composer run dev'</p>
